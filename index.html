Retirement Portfolio System

Personal app to monitor allocations, macro signals, and retirement readiness.

What this is

Personal, non‑commercial project integrating with financial data providers to monitor a retirement portfolio. Uses policy bands with glide‑path guardrails, overlays macro signals (inflation, debt, valuations, USD), and produces account‑aware rebalance suggestions across separate plans (Nationwide 457(b), FRS Investment Plan, Roth IRAs, HSA).

Scope
Read‑only aggregation & analytics for private use. No third‑party access.
Data Handling
Credentials are handled via provider OAuth flows; none are stored on this site.
Design
Core vs speculative sleeves, rule‑based tilts, monthly default cadence.
